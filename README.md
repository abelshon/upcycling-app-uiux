# ReVogue - UX Case Study

## Overview

ReVogue is an upcycling clothes application that encourages sustainable fashion by allowing users to redesign old clothes instead of discarding them.

## Problem

Fast fashion creates massive textile waste. Users lack an easy platform to upcycle clothing.

## Solution

ReVogue connects users with designers and provides inspiration, customization, secure ordering, and sustainable fashion guidance.
